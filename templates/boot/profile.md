# Profile - 用户配置

此文件可以编辑，用于配置个人偏好。修改后立即生效，无需重启。

## 基本信息

| 属性 | 值 |
|------|-----|
| 姓名 | [你的姓名] |
| 语言 | 中文 |
| 时区 | Asia/Shanghai |

## 偏好设置

### 回复风格
- 简洁回复 - 直接给出答案

### 通知设置
- 长时间任务时发送状态更新
- 错误时发送卡片消息

### 工作习惯

**工作目录：**
```
~/projects
```

**常用工具：**
- git
- npm / yarn / pnpm
- docker

## 常用命令

```bash
# 开发
npm run dev

# 测试
npm test

# 构建
npm run build
```

## 快捷短语

| 短语 | 操作 |
|------|------|
| `状态` | 显示当前项目状态 |
| `提交` | 执行 git commit |
| `推送` | 执行 git push |
| `测试` | 运行测试 |
