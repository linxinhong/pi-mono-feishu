# Profile - 用户配置

## 基本信息

| 属性 | 值 |
|------|-----|
| 姓名 | [你的姓名] |
| 语言 | 中文 |
| 时区 | Asia/Shanghai |

## 偏好设置

### 回复风格
- 简洁回复 - 直接给出答案

### 通知设置
- 长时间任务时发送状态更新
- 错误时发送卡片消息

## 快捷短语

| 短语 | 操作 |
|------|------|
| `状态` | 显示当前项目状态 |
| `提交` | 执行 git commit |
| `推送` | 执行 git push |
| `测试` | 运行测试 |

## 忽略规则

以下目录/文件不会被 AI 访问：

```
node_modules/
.git/
*.log
.env
*.secret
```
