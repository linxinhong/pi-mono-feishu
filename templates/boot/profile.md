## 快捷短语

可以定义一些快捷短语，AI 会识别并执行相应操作：

| 短语 | 操作 |
|------|------|
| `状态` | 显示当前项目状态 |
| `提交` | 执行 git commit |
| `推送` | 执行 git push |
| `测试` | 运行测试 |

## 忽略规则

以下目录/文件不会被 AI 访问：

```
node_modules/
.git/
*.log
.env
*.secret
```
